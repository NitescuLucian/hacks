
# hpn

## Why? 

I just need to convert nmap output in naabu format. (Host Port Nmap)

## Usage Example

```
hpn /tmp/nmap.xml
```

## Install

You can install using go:

```
go install -v github.com/NitescuLucian/hacks/hpn@latest
```
